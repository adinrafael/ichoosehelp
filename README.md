# I Choose Help

A professional, mobile-friendly website for help with OCD, trauma, and phobias.

## Features

- Responsive layout & animated menu
- Easy to add/edit content
- Contact form (Formspree integration)
- Ready for Vercel + GitHub deployment

3. **Deploy:**  
- Push the folder to GitHub

## Folder Structure

- `/app` — all page files
- `/components` — Header, Menu, Footer, ContactForm
- `/public` — logo, menu icon, background, favicon

## Editing Content

- Each page is a file in `/app` (e.g., `/app/about-us/page.jsx`)
- Edit text as you wish, using any code/text editor
- Add pages by copying and editing an existing file

## Assets

- `/public/logo.png` — site logo
- `/public/menu_logo.png` — menu icon
- `/public/background.jpg` — site background
- `/public/favicon.ico` — favicon (can use logo)
